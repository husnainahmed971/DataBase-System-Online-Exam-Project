# DataBase-System-Online-Exam-Project

## Description:
The Online Exam Project is a database-driven web app built with Django. It allows teachers to manage exams, questions, and results, while students can take tests. Designed to demonstrate database concepts.

## Tech Stack:

- Frontend: Django (HTML, CSS)
- Backend: Python, SQL

## How to Run:

git clone https://github.com/husnainahmed971/DataBase-System-Online-Exam-Project.git

cd DataBase-System-Online-Exam-Project

python manage.py runserver
